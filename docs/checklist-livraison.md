# CHECKLIST LIVRAISON — HAROBOZ Preview

## Par page — Vérifier AVANT le build

### SEO Technique
- [ ] H1 unique contenant le mot-clé principal
- [ ] Mot-clé principal dans les 100 premiers mots
- [ ] Méta titre ≤ 60 car. avec signal EEAT
- [ ] Méta description ≤ 155 car. avec CTA
- [ ] Slug court, sans stop words ni accents
- [ ] Balises Hn hiérarchiques (pas de saut H1 → H3)
- [ ] Schema FAQ sur les pages avec FAQ

### Maillage Interne
- [ ] Lien vers page mère dans les 200 premiers mots
- [ ] Pages sœurs linkées dans le corps
- [ ] Ancres variées (aucun doublon par URL)
- [ ] Max 5 liens sortants (8 pour pages piliers)
- [ ] Tous les liens pointent vers des pages existantes dans le preview

### Contenu & Copywriting
- [ ] Ton : rassurant, confidentiel, valorisant, local, premium
- [ ] Vocabulaire de marque utilisé (voir fiche-client.md)
- [ ] Aucun mot de la liste "À ÉVITER"
- [ ] Paragraphes courts (≤ 5 lignes)
- [ ] Mix de formats (texte + liste/tableau/encadré)
- [ ] FAQ 7-8 questions
- [ ] CTA visibles

### Design & HTML
- [ ] HTML conforme à la maquette
- [ ] Classes préfixées `hrb-` (pas de conflit futur WP)
- [ ] Responsive (mobile + desktop)
- [ ] Images avec alt descriptifs
- [ ] Pas de JS lourd / framework inutile
